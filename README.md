# What Is This Project? 
</br>
amalgichevdesign is a custom coded and designed website portfolio that displays projects related to Web Development and UI/UX Design, built to highlight
the necessary employable technical skills displayed by the creator. The tech stack for the website is HTML, CSS, JavaScript, Three.js alongside Cannon-es physics, Node.js, and Express.js.

</br>

# Features Included...
</br>

* Fully designed to be responsive for all screen sizes and to function with JavaScript turned off.

</br>

* Clean UI/UX design with various interactive elements that shows dynamic function.
  
</br>

* Links to other projects to showcase technical knowledge related to web development.

</br>

* Sections about the creator, experience, skills, projects and contact information.
